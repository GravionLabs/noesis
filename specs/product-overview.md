# Product Overview

Noesis is a self-hosted documentation context engine.

## Purpose
- Crawl external documentation sources.
- Store normalized documents and chunks in Postgres + pgvector.
- Expose search and retrieval through MCP tools and HTTP APIs.

## Supported client types
- GitHub Copilot CLI
- VS Code / Copilot Chat
- Any MCP-compatible client

## Core capabilities
- Semantic search over indexed documentation.
- Chunk retrieval by ID.
- Source management and import orchestration.
- Support for async crawling and embedding pipelines.
